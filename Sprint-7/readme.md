# 🧭 Guide to Sprint 7 — DOM Manipulation 🛒

> [!TIP]
> This is where **HTML, CSS, and JavaScript finally come together!**
> Make sure you've completed some HTML/CSS assignments first.

## Overview

In this sprint, you'll learn **DOM manipulation** by building a **shopping cart UI**. This is the bridge between your HTML/CSS skills and JavaScript — you'll make web pages interactive and dynamic.

**Real-world application:** Amazon, food delivery apps, any e-commerce site uses DOM manipulation to update the UI based on user actions.

## Learning Objectives

By the end of this sprint, you will be able to:

- ✅ Select HTML elements with `querySelector` and `querySelectorAll`
- ✅ Manipulate element content and attributes
- ✅ Add and remove CSS classes dynamically
- ✅ Handle user events (clicks, input, etc.)
- ✅ Create and insert new HTML elements
- ✅ Remove elements from the DOM
- ✅ Work with form inputs
- ✅ Use `localStorage` to persist data

## Structure

### 1. Key Exercises (`1-key-exercises/`)

Short, focused exercises with HTML files:

1. **Select elements** - querySelector basics
2. **Change content** - textContent, innerHTML
3. **Toggle classes** - classList methods
4. **Event listeners** - Click handlers
5. **Create elements** - Document.createElement

### 2. Mandatory Implement (`2-mandatory-implement/`)

Build shopping cart features:

1. `addItemToCart()` - Add product to cart
2. `removeItemFromCart()` - Remove product
3. `updateQuantity()` - Change item quantity
4. `calculateTotal()` - Sum cart total

### 3. Stretch Goals (`3-stretch/`)

Optional challenges:

- Complete shopping cart with localStorage
- Add animations and transitions
- Implement cart persistence across page loads

## Special Note

**This sprint includes HTML and CSS files!** You'll be working with actual web pages, not just JavaScript files.
